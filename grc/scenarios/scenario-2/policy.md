# Employee Offboarding Policy

## 1. Overview
When an employee or contractor leaves HaxNation, it is critical to ensure that all logical access to company resources is terminated promptly to minimize security risks.

## 2. Access Revocation SLA
- **Standard Termination**: All logical access (email, VPN, internal applications) must be revoked no later than **7 calendar days** from the employee's official last day of employment.
- **Immediate Termination (For Cause)**: Access must be revoked immediately, concurrent with the termination meeting.

## 3. Auditing
The IT Security department will conduct monthly audits cross-referencing HR departure logs with active directory systems to ensure SLA compliance. Any accounts active beyond the 7-day threshold must be flagged as a security incident.
